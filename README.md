## 📲 What this app does

Universal App Installer for Android helps you install apps in more than one format from one place. It supports:

- APK
- APKS
- XAPK
- APKM

It uses the native Android PackageInstaller, so it keeps the install flow close to what Android already knows. It also supports split app packages and OBB cache files for games and large apps.

## 📁 Supported package types

### APK
A standard Android app package. This is the most common format and usually installs in one step.

### APKS
A split app bundle package. The installer reads the package and installs the needed parts together.

### XAPK
A package format that can include the app plus extra data files. This works well for large games and apps that use OBB content.

### APKM
A package type used for bundled app installs. The app handles the install flow in the same place.

## 📦 OBB support

Some Android apps and games use OBB files for extra data. Universal App Installer can work with OBB cache files when they are packed with the app or placed next to it in the right folder structure. This helps when a game needs more than the main APK to run.

## 🧭 Troubleshooting

### The app does not open the file
Check that the file finished downloading. Then try again from your Downloads folder or file manager.

### Android blocks the install
Enable install permission for the app you used to open the file. Then try the install again.

### The package fails part way through
Make sure all split files are still in the same folder. Check that the file is not damaged, then download it again if needed.

### The app asks for storage access
Allow access so the installer can read the package file from your device.

### The game installs but does not launch
Check that the OBB data is in the correct place and that the package included all needed files.
