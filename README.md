# UAI (Package Installer Android)

Это сверхлегкий и универсальный установщик Android-пакетов, написанный с нуля на чистом **Kotlin** с пользовательским интерфейсом на **Jetpack Compose**.

Основная цель программы — предоставить крошечный по весу, но всеядный инструмент для установки составных пакетов и кэша в один клик напрямую через системный `PackageInstaller`, без рекламы, телеметрии и лишних C++ зависимостей.

### Ключевые особенности:
- **Размер APK**: Жесткая R8-оптимизация и использование нативных систем API позволили снизить вес приложения до фантастических **~800 КБ**.
- **Всеядность**: Установка стандартных файлов `.apk`, сплит-архивов `.apks` и `.apkm`, а также игр в формате `.xapk`.
- **Автоматический перенос OBB**: Для сложных `.xapk` архивов приложение само распознает `Base APK`, считывает его `packageName` и автоматически перемещает все файлы `.obb` напрямую в директорию `Android/obb/`.
- **Интерфейс**: Реализован стильный, глубоко-тёмный интерфейс с собственным встроенным файловым проводником без лишнего мусора, строго отсекающим любые ненужные файлы по формату.
- **Интеграция с Android**: Отсутствие громоздких фоновых процессов. Идеальная работа через `SessionParams.MODE_FULL_INSTALL`, поддержка процессоров всех типов (arm-v7, arm64, x86) и шторки уведомлений, исчезающей точно по окончанию бродкастов инсталлера.

### Как использовать?
1. Загрузите и запустите собранный `UAI.apk` на своем смартфоне.
2. При нажатии «Выбрать файл» выдайте запрашиваемые системой разрешения (`Установка из неизвестных источников` и `Доступ ко всем файлам`).
3. В открывшемся нативном проводнике найдите ваш файл `.apk`, `.apks`, `.xapk` или `.apkm`.
4. Дождитесь подготовки кэша, подтвердите системное диалоговое окно Android и программа установит всё за секунды!
